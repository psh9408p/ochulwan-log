# AGENTS.md

이 파일은 Codex가 리포지터리에서 작업할 때 참고할 짧은 지도입니다. 상세 정책과 기록은 `docs/` 아래 문서를 우선합니다.

## 핵심 원칙

- 과금 요소가 생길 수 있는 변경은 진행하지 않습니다.
- 카드 등록, 유료 플랜, 사용량 기반 과금, Telegram Stars/Paid Broadcasts는 금지합니다.
- 비용 관련 판단은 `docs/policies/no-cost-policy.md`를 기준으로 합니다.
- 외부 서비스 연결 전에는 무료 여부와 자동 과금 가능성을 먼저 확인합니다.

## 주요 문서

- `README.md`: 프로젝트 소개와 실행 방법
- `docs/policies/no-cost-policy.md`: 무과금 운영 정책
- `docs/superpowers/specs/2026-07-03-telegram-attendance-design.md`: 초기 제품/기능 설계
